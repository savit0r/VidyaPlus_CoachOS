# Mintlify Design System

## Core Aesthetic
- Developer-grade information density.
- Cinematic, atmospheric, and highly legible.
- Professional, modern, minimal SaaS look.

## Colors
- **Brand Green**: `#00D4A4`
- **Brand Green Deep**: `#00AA85`
- **Testimonial Orange**: `#F26B4F`
- **Canvas (Light)**: `#FFFFFF`
- **Canvas (Dark)**: `#000000`
- **Surface**: `#F9FAFB` (Light), `#111111` (Dark)
- **Ink (Text)**: `#111827` (Light), `#FFFFFF` (Dark)
- **Steel (Muted Text)**: `#7C8794`
- **Hairline (Borders)**: `#E5E7EB` (Light), `#1F1F1F` (Dark)

## Typography
- **Primary (Sans)**: Inter, system-ui, sans-serif
- **Secondary (Mono)**: Geist Mono, monospace

## Component Styles
### Pricing Cards
- Standard tier: solid canvas bg, 1px hairline border, 12px rounded corners, 32px internal padding
- Featured tier: 2px solid brand-green border, soft brand-tinted shadow rgba(0,212,164,0.08) 0px 8px 24px
- Price display: 56px font-weight-600 mono, tight -1.5px letter-spacing
- Feature list items: 14px body with green checkmark icons, 8px vertical spacing

### Buttons
- Fully rounded (9999px), 10px 20px padding
- Primary: black bg, white text
- Brand accent: brand-green bg, black text
- Secondary: transparent bg, hairline border

### Badges
- 11px uppercase, 0.5px letter-spacing
- Subtle 10% brand-green bg with 20% border

## Spacing
- Card padding: 32px
- Section gap: 96px (marketing), 64px (comparison)
- Feature list item gap: 8px

## Elevation
- Flat by default (no shadows on standard cards)
- Featured pricing: brand-tinted glow shadow
- Hero mockup: deep diffuse drop shadow

## Shapes
- Cards: 12px border-radius
- Buttons: 9999px (full pill)
- Code blocks: 8px
- Inputs: 8px
